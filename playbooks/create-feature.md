# Create Feature Playbook

1. Define business requirements.
2. Create feature folder.
3. Define routes.
4. Create domain models.
5. Create DTOs.
6. Implement datasource.
7. Implement repository.
8. Create Riverpod controller.
9. Build screens.
10. Build reusable widgets.
11. Add loading, empty, and error states.
12. Write unit tests.
13. Write widget tests.
14. Update documentation.
15. Perform accessibility review.
16. Submit for code review.
